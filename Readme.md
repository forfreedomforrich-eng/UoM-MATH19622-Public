## 🐍 Running the Code
We use **Python** for our statistical analysis and data visualisation. To ensure reproducibility, we use the `uv` package manager.

1. **Clone the repo:** `git clone https://github.com/antRthorn/UoM-MATH19622-Public.git`
2. **Run a script:** `uv run path/to/script.py`
